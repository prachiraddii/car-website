🚗 Car Website

Welcome to our car website! This project is a modern and interactive platform designed to showcase different cars, their features, and provide users with a smooth browsing experience.

📌 About

This website is built using HTML, CSS, and JavaScript. It allows users to explore various cars, view details, and interact with a clean and responsive interface. The goal of this project is to combine design and functionality to create an engaging automotive browsing experience.

🔗 Live Demo



📁 Features

* Responsive design for mobile, tablet, and desktop
* Interactive UI with smooth animations
* Car listings with detailed information
* Login/Signup functionality
* Easy navigation and user-friendly layout

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
